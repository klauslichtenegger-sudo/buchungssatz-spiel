# Buchungssatz bauen

Eine kleine React-Übung für Buchungssätze mit Soll, Haben, Umsatzsteuer, Vorsteuer und Kontoabschluss.

## Lokal starten

```bash
npm install
npm run dev
```

## Veröffentlichen

Das Projekt ist für GitHub Pages vorbereitet. Der Produktions-Build entsteht mit:

```bash
npm run build
```
